Política de Privacidade — postar-videos

postar-videos é uma ferramenta pessoal de linha de comando (Python), usada exclusivamente pelo seu único desenvolvedor e usuário, Lucas Prado Luchi, para enviar vídeos ao seu próprio canal do YouTube por meio dos Serviços de API do YouTube (YouTube Data API v3).

  • A ferramenta não tem outros usuários além do próprio desenvolvedor e não é distribuída ao público.
  • Não coleta, não armazena e não compartilha dados de terceiros. Os únicos dados tratados são os arquivos de vídeo e metadados (título, descrição, tags) criados pelo próprio usuário, e o token de autorização OAuth do Google, armazenado apenas no computador pessoal do usuário.
  • O acesso à conta do YouTube usa o consentimento OAuth oficial do Google e pode ser revogado a qualquer momento em myaccount.google.com/permissions.
  • Nenhuma informação da API do YouTube é retida além do necessário para concluir cada envio, nem repassada a qualquer terceiro.
  • O uso dos Serviços de API do YouTube está sujeito à Política de Privacidade do Google e aos Termos de Serviço do YouTube.

English summary

postar-videos is a personal command-line tool used solely by its single developer/user to upload videos to his own YouTube channel via the YouTube Data API v3. It has no other users, collects no third-party data, stores nothing beyond the user's own video files, metadata and the OAuth token kept locally on his personal computer, and shares no data with anyone. YouTube API access can be revoked at any time at Google security settings.
