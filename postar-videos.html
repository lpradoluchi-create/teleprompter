postar-videos

Ferramenta pessoal de linha de comando que publica vídeos curtos no canal do YouTube do seu único usuário e desenvolvedor, por meio dos Serviços de API do YouTube (YouTube Data API v3).

O fluxo é simples: o vídeo é produzido e revisado manualmente; a ferramenta cuida apenas do envio com título, descrição e agendamento. Uso estritamente pessoal — a ferramenta não é distribuída, não tem outros usuários e não coleta dados de terceiros.

Este produto usa os Serviços de API do YouTube. Ao utilizá-lo, você concorda com os Termos de Serviço do YouTube.
